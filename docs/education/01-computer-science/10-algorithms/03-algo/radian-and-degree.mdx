## Definition

<Tabs>
  <TabItem value="definition" label="Definition">
    It's used to convert between radians and degrees, providing flexibility in mathematical calculations involving angles
  </TabItem>
  <TabItem value="how" label="Explanation">
    Conversion factors: `π` radians equals 180 degrees, and 1 radian equals approximately 57.2958 degrees.
  </TabItem>
  <TabItem value="guidance" label="Guidance">
    - To convert radians to degrees
      - multiply the radian value by `180/π`
    - To convert degrees to radians
      - multiply the degree value by `π/180`
  </TabItem>
  <TabItem value="tips" label="Tips">
    - ensure precision in conversion by using accurate mathematical constants like `π`
  </TabItem>
</Tabs>

## Practice

<Tabs>
  <TabItem value="practice" label="Practice">
    ```python
    convertRadiansToDegrees(radians):
      // Multiply radians by the conversion factor to get degrees
      degrees = radians * (180 / π)
      return degrees

    convertDegreesToRadians(degrees):
      // Multiply degrees by the conversion factor to get radians
      radians = degrees * (π / 180)
      return radians
    ```
