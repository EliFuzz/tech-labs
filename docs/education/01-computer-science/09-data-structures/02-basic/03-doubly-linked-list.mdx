## Definition

<Tabs>
  <TabItem value="short" label="Short">
    **Doubly Linked List** is a type of data structure in which each node contains a data part and 2 pointers, one pointing to the next node and another pointing to the previous node in the sequence.

    <details>
      <summary>Simplified</summary>

      **Doubly Linked List** is like a game of hopscotch. Each square, or "node", holds an item and information about the next and previous squares. You start at the "head" and can move forward or
      backward to the "tail". You can also add or remove squares during the game.
    </details>

  </TabItem>
  <TabItem value="detailed" label="Detailed">
    **Doubly Linked List** in computer science is a sequence of nodes linked in both directions. Each node has two links pointing to its previous and next nodes. The list starts and ends with a
    terminator, usually a sentinel node or null, aiding in list traversal. If there's only one sentinel node, the list forms a circular link through it. Essentially, it's like having two singly linked
    lists with the same data but in reverse orders.
  </TabItem>
</Tabs>

            delete_head():
              if head == ø:
                return ø

              if head.next != ø:
                head = head.next
                head.previous = ø
              else:
                tail = ø

          <td>Delete Node</td>
          <td>
            ```
            delete_node(value):
             current = head
             while current != ø:
               if current.value == value:
                 if current == head:
                   head = current.next
                   head.previous = ø
                 else:
                   current.previous.next = current.next
                   current.next.previous = current.previous
                 return current
               current = current.next
            ```

            delete_tail():

            reverse():

              while current_node != ø:
                next_node = current_node.next
